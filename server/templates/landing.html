MRI Project

This application allows users to load and view mha or nii files. To use this application, load a file by clicking "Choose File" on the left, then click "Load". The viewer then will appear. To change the slice, the user can either move the slider or change the number in the text input. Double click the image in order to use that slice for the pre-processing and segmentation sections. Click the next button to get to the pre-processing section. For the pre-processing section, click a button in order to apply that filter. Click the next button to get to the segmentation section or click the back button to get back to the views section. For the segmentation, click the apply watershed button in order to apply multiple filters. Click the back button to get back to the pre-processing section. To change the current mha or nii file, click the current file name in the top left and click the file desired.
